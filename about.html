# 关于我们

# 我们是做什么的？

PC-helper电脑社是一个公益技术类团体，在张克荣书记和指导老师陈洁的带领下，社团成立一年以来快速稳定发展，从最初的八个人发展到现在三百多人的大集体。我们坚持以正确、高效的理念来管理社团，贯彻“普及电脑知识，提供义务服务”的宗旨，每星期开设培训课程，每月开展全校电脑义修活动，包括电脑软、硬件使用、维修及维护，windows系统的封装及笔记本的内部结构组成，以理论+实践的方式快速掌握和加强巩固及其基本要领所在，达到事半功倍的效果，致力于打造一个重学术、抓实践、强交流的精英团体。

我们的定位在于公益，意在帮助和教与校内师生自己动手解决一些力所能及的电脑故障问题，减少电脑发生小故障时出去维修带来的经济损失和时间浪费的诸多弊端。每个学期，我们都会定期开设电脑软、硬件知识培训课程，并且定期在学校北门、厚德楼广场、凤翔餐厅旁设立电脑义修点，走近同学们的生活，与同学们进行更好的学习与交流。同时，我们也设有其它故障报修的方式，如微信公众号，微博等传媒工具，以便给大家提供更好更快捷的服务，提升自身对知识技能的掌握程度。

计算机学院辅导员陈洁老师是电脑社的指导老师，陈洁老师拥有丰富的管理经验，给我们传授自身感触点，通过“以点对事”的方式，在多方面传授给我们宝贵的实战经验。对于社团工作进展和安排，老师给予了极大的肯定，鼓励我们抛开旧思想，往新社团、新方向发展，以多元化的方式，最大化的去接受和学习各种知识技能，正所谓技多不压身，这样才能符合信息时代的人才需求。

# 我们能做什么？

# Office办公软件应用

包过word文字处理，excel表格，以及PPT的应用。主要针对计算机二级考试和日常使用到的一些功能技巧，方便我们在办公使用的时候节约时间。

# PS使用入手到精通

Photoshop是专门处理图片的一款软件，其功能强大，容易上手学习，符合大众，我们先从PS的界面开始解释到利用各个功能去实现完成一张图片，学会了运用控件也就基本学会了PS。

# Windows系统封装

电脑系统是我们使用频率较高的一项，但是不可避免的由于一些误操作可能会导致电脑系统无法正常启动，如果使用官方镜像重装，系统过于臃肿，特别是针对一些配置不高的老电脑将非常不友好。我们会将其进行修改，裁剪，优化等操作，使其变成一个我们使用起来可以稳定工作的一个平台。

# 电脑软、硬件维护

当我们电脑使用一段时间过后会出现，电脑卡顿或是蓝屏频繁，某些功能丢失，电脑无法开机等常见问题，我们可以优化您的电脑。使其可以发挥在一个反应快速，使用流畅的状态。并且出现问题后我们会分析是什么原因造成的，并将其修复以及教您如何避免再次出现同样情况。

# 网络故障分析和处理

网络是让我们的设备连接到互联网中，但有时候小小的网络故障很让人头疼，想想你吃鸡的时候被人突脸的时候突然网络延迟检测，所以学会一些网络故障维修是很有帮助的。

# 网站建设

网站建设对于现在来说不是一个很难的项目，并且入门成本低，几块钱买域名在自己的电脑上建立起服务器。但是其涉及到基础的HTML网页框架，CSS美化网页，JS动态效果等语言，让我们去接触学习这样语言，会让我们的思维变的更加严谨。
